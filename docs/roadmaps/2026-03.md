# Fictotum — March 2026 Roadmap

## North Star
Fictotum becomes a product people want to share — AI-discovered connections grounded in graph data, automated content growth, and visual richness that makes every figure page worth exploring.

### M2: Discovery Agent — Complete
- **Why it matters**: The "wow" feature. AI-discovered connections grounded in graph data. This is what makes Fictotum different from a Wikipedia sidebar. Trails competitive analysis (Feb 20) validated the concept — but Fictotum's typed graph is a structural advantage over unstructured text.
- **Acceptance criteria**:
  - [x] `lib/connection-scoring.ts` — graph-only signals (hop distance, cross-era surprise, sentiment divergence, property completeness)
  - [x] "Discover Connections" section on figure detail pages (on-demand, not precomputed)
  - [x] Claude Sonnet 4.6 narration of discovered connections with citations to graph data — implemented with graceful fallback when ANTHROPIC_API_KEY not set
  - [x] Connections feel grounded in historical fact, not arbitrary (the Trails critique) — path summaries cite shared media, narration prompt requires graph-data grounding
  - [x] Test on 5+ figures with known interesting connections — tested Napoleon, Gandhi (orphan), Shakespeare, Neil Armstrong, Henry VIII, Jesus, Julius Caesar
- **Linear tickets**: FIC-120, FIC-118
- **Key files**: `web-app/lib/connection-scoring.ts`, `web-app/app/api/figure/[id]/discover/route.ts`, `web-app/components/DiscoverConnections.tsx`, `web-app/app/figure/[id]/page.tsx`
- **Dependencies**: FIC-120 (scoring) must be built before FIC-118 (discovery agent)
- **Notes**: Performance guard skips expensive shortestPath query for figures with >12 portrayals. ANTHROPIC_API_KEY needed in web-app/.env.local for AI narration (without it, graph-scored connections still display with path summaries).

## Risks
- **Gemini quota limits** could bottleneck both M3 and M4. Mitigation: check quotas before starting, prioritize M3 over M4 if constrained.
- **FIC-148 could be deeper than expected**. Mitigation: timebox to 1 day, escalate to Vercel support if unresolved.
- **Discovery agent quality** — AI connections that feel shallow will hurt credibility (the Trails/HN critique). Mitigation: FIC-120 scoring before FIC-118 generation, ground everything in graph data with citations.
